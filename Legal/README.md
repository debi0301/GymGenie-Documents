## ⚠️ 必須の編集項目

これらの文書を公開する前に、以下の情報を実際の情報に置き換えてください：

### 全ファイル共通
- `[運営会社名を記載]` → 実際の会社名または個人名
- `[代表者名を記載]` → 代表者の氏名
- `[住所を記載]` → 実際の住所
- `[郵便番号]` → 実際の郵便番号
- `[電話番号を記載]` → 実際の電話番号
- `gymgenie.support@example.com` → 実際のサポートメールアドレス

### 特定商取引法に基づく表記
- サブスクリプション価格（例: ¥980/月）→ 実際の販売価格

## Web公開の手順

これらの文書は、Webページとして公開する必要があります。以下のいずれかの方法で公開してください：

### 方法1: GitHub Pages（無料・推奨）

1. GitHubリポジトリを作成
2. `Legal/` フォルダをコミット
3. GitHub Pages を有効化
4. URLが自動生成されます（例: https://username.github.io/gymgenie-legal/利用規約.html）

### 方法2: 自社ウェブサイト

既にウェブサイトをお持ちの場合：
- `/terms/` - 利用規約
- `/privacy/` - プライバシーポリシー
- `/commercial-transaction/` - 特定商取引法に基づく表記

### 方法3: 無料ホスティングサービス

- Netlify (https://www.netlify.com/)
- Vercel (https://vercel.com/)
- Cloudflare Pages (https://pages.cloudflare.com/)

## よくある質問

### Q: 個人開発者でも会社名が必要？
A: 個人の場合、個人名でも可能です。ただし、屋号（例: GymGenie開発室）を使用することもできます。

### Q: 住所を公開したくない場合は？
A: 日本の法律では、特定商取引法により販売事業者の住所表示が必須です。バーチャルオフィスの利用も検討できます。

### Q: プライバシーポリシーは英語も必要？
A: 日本国内向けアプリの場合、日本語のみで問題ありません。海外展開する場合は英語版も作成してください。
