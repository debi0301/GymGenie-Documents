# App Store 申請前チェックリスト

このチェックリストは、App Storeに申請する前に確認すべき項目をまとめたものです。

---

## 📱 アプリの準備

### ビルド・テスト
- [ ] **最新版のビルドが完了**している
- [ ] **実機でテスト済み**（クラッシュしない）
- [ ] **シミュレーターでテスト済み**（複数デバイス）
- [ ] **全ての主要機能が動作**する
- [ ] **APIキーが正しく設定**されている
- [ ] **ネットワークエラー時の挙動**を確認済み
- [ ] **メモリリークがない**ことを確認

### 法的要件
- [ ] **利用規約を作成**し、Webで公開済み
- [ ] **プライバシーポリシーを作成**し、Webで公開済み
- [ ] **特定商取引法に基づく表記を作成**し、Webで公開済み
- [ ] **医療免責事項**がアプリ内に表示される
- [ ] **Settings画面のリンク**を実際のURLに変更済み

### メタデータ
- [ ] **運営者情報**（会社名/個人名、住所、電話番号）を記入済み
- [ ] **サポートメールアドレス**を設定済み（実際に受信できる）
- [ ] **著作権表記**を記入済み

---

## 🎨 App Store Connect

### アプリ情報
- [ ] **アプリ名**を決定（30文字以内）
- [ ] **サブタイトル**を作成（30文字以内）
- [ ] **プロモーションテキスト**を作成（170文字以内）
- [ ] **説明文**を作成（4000文字以内）
- [ ] **キーワード**を選定（100文字以内）
- [ ] **カテゴリ**を選択（主: ヘルスケア/フィットネス）
- [ ] **年齢制限**を設定（4+）

### URL設定
- [ ] **サポートURL**を設定
- [ ] **プライバシーポリシーURL**を設定（必須）
- [ ] **マーケティングURL**を設定（オプション）
- [ ] 全てのURLが**実際にアクセス可能**か確認

### ビジュアル素材
- [ ] **アプリアイコン**（1024x1024 px）をアップロード
  - 透過なし
  - 角丸なし（自動で適用される）
  - 高解像度
- [ ] **スクリーンショット**を5〜10枚作成
  - 6.7インチ: 1290 x 2796 px（必須）
  - テキストオーバーレイあり
  - 魅力的な画面を選定
- [ ] **アプリプレビュー動画**を作成（オプション、推奨）
  - 15〜30秒
  - 縦向き
  - 音声なしでも理解できる内容

## 🔐 プライバシー・セキュリティ

### App Store Privacy Details
- [ ] **データ収集の種類**を申告
  - 健康とフィットネス: 体重、身長、ワークアウト情報
  - 使用状況データ: アプリの利用状況
  - 診断: クラッシュデータ
- [ ] **データの使用目的**を選択
  - アプリ機能
  - 分析
  - 製品のパーソナライゼーション
- [ ] **データの追跡**について申告
  - 現在は追跡なし
- [ ] **サードパーティSDK**の開示
  - OpenAI API（プライバシーポリシーに記載済み）

### セキュリティ対策
- [ ] **HTTPS通信**のみ使用
- [ ] **APIキーの保護**（ハードコード削除済み）
- [ ] **ユーザーデータの暗号化**
- [ ] **脆弱性テスト**を実施

## 🚀 申請

### ビルドアップロード
- [ ] **Xcodeでアーカイブ**作成
  - Product → Archive
- [ ] **App Store Connectにアップロード**
  - Organizer → Distribute App → App Store Connect
- [ ] **ビルドが処理完了**するまで待機（30分〜2時間）

### 最終確認
- [ ] **全ての必須項目**が入力済み
- [ ] **スクリーンショット**が全デバイスサイズにアップロード済み
- [ ] **ビルドを選択**
- [ ] **リリースタイプ**を選択
  - 手動リリース（推奨）
  - 自動リリース
- [ ] **価格設定**を確認
  - アプリ本体: 無料
  - サブスクリプション: 価格設定済み

### 申請送信
- [ ] **「審査に提出」ボタン**をクリック
- [ ] **確認メールを受信**
- [ ] **審査状況を定期的に確認**

---

## ⏰ 審査後のスケジュール

### 審査期間
- 通常: 24〜48時間
- 混雑時: 最大7日間

### 審査中にできること
- [ ] マーケティング素材の準備
  - SNS投稿用画像
  - プレスリリース
  - ランディングページ
- [ ] サポート体制の準備
  - FAQページ作成
  - サポートメール監視
- [ ] 分析ツールの準備
  - Firebase Analytics（今後）
  - App Store Connect Analytics

### 承認後
- [ ] **「リリース」ボタン**をクリック（手動リリースの場合）
- [ ] App Storeで**アプリが公開**されたことを確認
- [ ] **SNSで告知**
- [ ] **友人・家族に共有**
- [ ] **レビュー依頼**メールを送信（適切なタイミングで）

### リジェクトされた場合
- [ ] **リジェクト理由**を確認
- [ ] **必要な修正**を実施
- [ ] **再テスト**
- [ ] **再申請**
