# App Store スクリーンショット構成案

App Storeに掲載するスクリーンショットの構成案です。

---

## 📐 必要なサイズ

### iPhone
- **6.7インチ（iPhone 15 Pro Max等）**: 1290 x 2796 px（必須）
- **6.5インチ（iPhone 14 Plus等）**: 1242 x 2688 px（必須）
- **5.5インチ（iPhone 8 Plus等）**: 1242 x 2208 px

### iPad（オプション）
- **12.9インチ iPad Pro**: 2048 x 2732 px

**推奨**: 最大サイズ（6.7インチ）で作成し、自動調整させる

---

## 📸 スクリーンショット構成（最大10枚）

App Storeでは、最初の3枚が最も重要です。ユーザーは通常3枚目まで見て判断します。

### 🎯 スクリーンショット 1: **ヒーロー画像（最重要）**
**目的**: 一瞬でアプリの価値を伝える

**画面**: ウェルカム画面 または プラン生成完了画面

**テキストオーバーレイ**:
```
AIがあなた専用の
ワークアウトプランを作成

🤖 16の質問に答えるだけ
📊 完全カスタマイズ
💪 自宅でもジムでもOK
```

**デザイン**:
- 背景を少し暗くして、テキストを目立たせる
- GymGenieロゴを配置
- 黄金色のアクセントカラーを使用

---

### 📊 スクリーンショット 2: **プラン表示画面**
**目的**: 具体的な機能を見せる

**画面**: WorkoutPlanDetailView（7日間のプラン）

**テキストオーバーレイ**:
```
あなただけの7日間プラン

✅ 目標に合わせた種目選定
✅ レベルに応じたセット・レップ数
✅ 休息日も自動設定
```

**ポイント**:
- 実際のワークアウトプランが表示されている状態
- 「月曜日」「火曜日」などの曜日が見える
- エクササイズ名と詳細が確認できる

---

### 💪 スクリーンショット 3: **ワークアウトセッション画面**
**目的**: 使用中の体験を伝える

**画面**: WorkoutSessionView（セット記録中）

**テキストオーバーレイ**:
```
シンプルで使いやすい記録

⏱️ レストタイマー自動表示
📝 前回の記録を自動プリフィル
✨ タップするだけで完了
```

**ポイント**:
- ベンチプレスなどの人気種目を表示
- セット数（例: 3/4セット完了）が見える
- 重量・回数の入力フィールドが表示されている

---

### 📈 スクリーンショット 4: **進捗グラフ画面**

## 🎨 デザインガイドライン

### カラースキーム
- **背景**: ダークモード（#000000, #0A0A0A）
- **アクセント**: ゴールド（#F5A623）
- **テキスト**: ホワイト（#FFFFFF）、グレー（#9CA3AF）

### テキストオーバーレイ
- **フォント**: ヒラギノ角ゴシック、Noto Sans JP
- **フォントサイズ**: タイトル 28-32pt、本文 16-18pt
- **影**: テキストに薄い影をつけて読みやすく
- **位置**: 画面上部または下部（UI要素に被らない）

### アイコン
- SF Symbols を活用
- 絵文字で視覚的にわかりやすく

## 📸 スクリーンショット撮影手順

### 方法1: Xcode Simulator（推奨）
1. Xcode でシミュレーターを起動（iPhone 15 Pro Max）
2. 該当画面に遷移
3. `⌘ + S` でスクリーンショット保存
4. デスクトップに保存される

### 方法2: 実機で撮影
1. 実機でアプリを起動
2. 該当画面で「音量上ボタン + サイドボタン」
3. 写真アプリから取得

### 方法3: Xcode Cloud
- App Store Connect から直接スクリーンショットを撮影・編集

## 🖼️ スクリーンショット編集

### 推奨ツール

#### 無料ツール
1. **Canva** (https://www.canva.com/)
   - テンプレートあり
   - テキストオーバーレイが簡単

2. **Figma** (https://www.figma.com/)
   - プロフェッショナルな編集
   - チーム共有可能

3. **Apple Preview（プレビュー）**
   - Mac標準アプリ
   - 簡単な編集に

#### 有料ツール（オプション）
1. **Adobe Photoshop**
   - プロレベルの編集

2. **Sketch**
   - UIデザイナー向け

3. **Screenshot Creator** (App Store)
   - App Store スクリーンショット専用アプリ

## 📝 チェックリスト

撮影・編集前:
- [ ] シミュレーター/実機を準備
- [ ] アプリにサンプルデータを入力（ワークアウト履歴、プランなど）
- [ ] 最適な画面を選定

撮影:
- [ ] iPhone 15 Pro Max (6.7インチ) でスクリーンショット撮影
- [ ] 全画面（ステータスバー含む）で撮影
- [ ] 最低5枚、最大10枚撮影

編集:
- [ ] テキストオーバーレイを追加
- [ ] ブランドカラー（ゴールド）を使用
- [ ] 読みやすいフォント・サイズ
- [ ] UI要素に被らないよう配置

最終確認:
- [ ] 解像度が正しい（1290 x 2796 px）
- [ ] ファイル形式がJPEGまたはPNG
- [ ] ファイルサイズが500KB以下（1枚あたり）
- [ ] 全画面表示で見やすいか確認
- [ ] 誤字脱字がないか確認

## 🚀 アップロード

1. App Store Connect にログイン
2. 「マイApp」→「GymGenie」を選択
3. 「App Store」タブ → 「スクリーンショット」
4. デバイスサイズを選択（6.7インチ）
5. ドラッグ&ドロップでアップロード
6. 順番を並び替え（重要な順に）

---

## 💡 ヒント

- **最初の3枚が最重要**: ユーザーは最初の3枚で判断
- **機能より価値を見せる**: 「何ができるか」より「何が得られるか」
- **視覚的に魅力的に**: カラフルなグラフ、実際のデータ
- **テキストは最小限**: 画像で伝わる部分は画像で
- **競合を研究**: 人気フィットネスアプリのスクリーンショットを参考に
