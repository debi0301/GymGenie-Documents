# App Store 申請準備資料

このフォルダには、GymGenieをApp Storeに申請するための資料がまとめられています。

---

## 📁 ファイル一覧

### 1. アプリ説明文.md
App Store Connect で使用する以下の情報が含まれています：
- **アプリ名**（30文字以内）
- **サブタイトル**（30文字以内）
- **プロモーションテキスト**（170文字以内）
- **説明文**（4000文字以内）
- **キーワード**（100文字以内）
- **審査用メモ**

### 2. スクリーンショット構成案.md
App Storeに掲載するスクリーンショットの構成案：
- **必要なサイズ**: 1290 x 2796 px（iPhone 6.7インチ）
- **スクリーンショット構成**: 10枚の構成案
- **デザインガイドライン**: カラー、フォント、レイアウト
- **撮影・編集手順**: シミュレーターでの撮影方法、編集ツール

### 3. 申請前チェックリスト.md
App Store申請前に確認すべき項目の完全リスト：
- **アプリの準備**: ビルド、テスト、法的要件
- **App Store Connect設定**: メタデータ、URL、ビジュアル素材
- **App内課金設定**: サブスクリプション商品
- **プライバシー・セキュリティ**: データ収集の申告
- **最終テスト**: 機能テスト、UI/UXテスト
- **申請手順**: ビルドアップロード、審査送信

### ステップ4: スクリーンショットの作成

`スクリーンショット構成案.md` を参考に、スクリーンショットを作成してください：

1. **シミュレーターで撮影**（iPhone 15 Pro Max）
2. **Canva/Figmaでテキストオーバーレイを追加**
3. **5〜10枚作成**（最初の3枚が最重要）
4. **サイズ確認**: 1290 x 2796 px

---

### ステップ5: App Store Connectに登録

1. **Apple Developer Program登録**（年間¥11,800）
   - https://developer.apple.com/programs/

2. **App Store Connectでアプリ作成**
   - https://appstoreconnect.apple.com/

3. **メタデータ入力**
   - `アプリ説明文.md` の内容をコピー&ペースト

4. **スクリーンショットアップロード**

5. **App内課金設定**
   - 月額プラン: ¥980
   - 年額プラン: ¥9,800

6. **プライバシー詳細を申告**

7. **ビルドをアップロード**
   - Xcode → Product → Archive → Distribute

8. **審査に提出**

## ⚠️ 重要な注意事項

### 必ず実施すること

1. **法的文書のURL更新**
   - example.com のままだと審査で却下されます

2. **運営者情報の記入**

4. **医療免責事項の確認**
   - アプリ起動時に表示されることを確認

5. **サブスクリプションのテスト**
   - サンドボックス環境で購入フローをテスト

## 🎯 審査を通過するためのポイント

### 1. 完全性
- 全ての機能が正常に動作すること
- クラッシュしないこと
- 説明文と実際の機能が一致すること

### 2. 法的要件の遵守
- プライバシーポリシーが公開されていること
- 医療免責事項が表示されること
- サブスクリプション情報が明示されていること

### 3. デザイン品質
- UIが洗練されていること
- スクリーンショットが魅力的であること
- アプリアイコンが高品質であること

### 4. ユーザー価値
- 明確な価値提案
- 使いやすいUI/UX
- 実際に役立つ機能
